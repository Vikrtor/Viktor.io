Объвление¶

    При выборе объвление будут доступны действия :
    • Добавить в избранное(дейст.1)
    • Связаться(действ.2)

    Инструкции для каждого действия представлены далее.

    Добавить в избранное(дейст.1)

    С помощью действия добавить в избранное можно будет добавить объвления, чтобы в дальнешем рассмотреть лучшие варианты в одном месте функционал регистрации описан ниже:

    Добавление ¶

  • Перейдите на понравившиеся вам объвление
  • Нажмите на кнопку "Добавить в избраннные"

    • Удаление¶

  • Перейдите в "Профиль"
  • Перейдите в "Избранные"
  • Выберите объвление которое хотите удалить из раздела
  • Нажмите на кнопку "Убрать из избранных"

    • Обратная свзяь(действ.2) ¶

    Действие позволяеь связаться с продавцом квартиры, чтобы договориться о осмотре квартиры:

    Связаться¶

  • Перейдите на понравившиеся вам объвление
  • Нажмите кнопку "Связаться"
  • Свяжитесь с продавцем по данным которым он оставил

  • Вернуться к списку раздела
