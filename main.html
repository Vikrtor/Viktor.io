Главная страница¶

    На главном окне будет расположены меню с несколькими разделами:
    • Сортировка(раз.1)
    • Поиск(раз.2)
    • Разместить объявление(раз.3)

    Инструкции для каждой из кнопки представлены далее.

    Сортировка (раз.1)

    С помощью сортировки можно будет отсортировать объвление по выбранным критериям, функционал сортровки описан ниже:

    Поиск по критериям¶

  • Нажмите рядом с необходимым критерием
  • Введите минимальную и максимальную стоимость квартиру
  • Укажите районы (микрорайоны) где хотели бы проживать
  • Нажмите кнопку "Cортировать"

    • Сброс критериев¶

  • Нажмите кнопку "Сброс" для сброса критериев

    • Поиск (раз.2 ) ¶

    С помощью поиска можно будет искать объвления по названию, адресу и т.д

    Использование поиска¶

  • Наведитесь в поле ввода
  • Наберите на клавитатуре по то что хотите найти
  • Нажмите кнопку "Поиск"

    • Сброс поиска¶

  • Наведитесь в поле ввода
  • Нажмите на крестик чтобы сбросить запрос

    • Разместить объявление (раз.3 ) ¶

    С помощью этой кнопки пользователь может выставить свою квартиру на аренду или продажк

    Оставить объявление¶

  • Нажмите на кнопку разместить объвление
  • Напишите об вашей квартиры
  • Заполните данные о квартире
  • Укажите стоимость сделки
  • Приложите скриншоты квартры
  • Нажмите кнопку "Выложить объвление"

    • Удалить объявление¶

  • Нажмите на кнопку разместить объвление
  • Нажмите на кпопку "Мои объявления"
  • Выберите объявление которое хотите удалить
  • Нажмите кнопку "Удалить объявление"

    • Отредактировать объявление¶

  • Нажмите на кнопку разместить объвление
  • Нажмите на кпопку "Мои объявления"
  • Выберите объявление которое хотите отредактировать
  • Измените данные о квартире
  • Нажмите кнопку "Сохранить изменения"

  • Вернуться к списку раздела
