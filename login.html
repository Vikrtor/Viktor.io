Авторизация¶

    При входе на сайт вход в систему можно выполнить двумя способавми:
    • Регистрация(сп.1)
    • Вход(сп.2)

    Инструкции для каждой из кнопки представлены далее.

    Регистрация (сп.1)

    С помощью регистраци можно авторизироваться на сайте если вы еще не были зарегистрированы на сайте, функционал регистрации описан ниже:

    Регистрация на сайте ¶

  • Нажмите на кнопку "Регистрация"
  • Введите логин
  • Придумайте и введите пароль
  • Введите дополнительные данные
  • Нажмите кнопку "Ок"

    • Выход¶

  • Нажмите на иконку профиля
  • Нажмите на кнопку "Выйти"
  • Подтвердите выход, нажмите кнопку "Да"

    • Вход на сайте(сп.2 ) ¶

    С помощью входа можно будет авторизироваться используя логин и пароль, функционал регистрации описан ниже:

    Вход¶

  • Нажмите на кнопку "Вход"
  • Введите свой логин
  • Введите свой пароль
  • Нажмите кнопку "Ок"

    • Выход¶

  • Нажмите на иконку профиля
  • Нажмите на кнопку "Выйти"
  • Подтвердите выход, нажмите кнопку "Да"

  • Вернуться к списку раздела
